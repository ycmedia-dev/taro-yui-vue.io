# 搜索栏 y-search-bar

输入关键字，并筛选条件进行搜索,支持插槽，有蓝色和白色款， 默认蓝色
[YFormElType.DATE_PICKER, YFormElType.DATE_TIME_PICKER, YFormElType.CUSTOM_SELECTOR, YFormElType.SELECTOR]支持清除配置，默认可清除

### 组件名称

y-search-bar

### 依赖组件

1. y-input-box
2. y-date-picker
3. y-date-time-picker
4. y-custom-selector
5. y-form-item-picker
